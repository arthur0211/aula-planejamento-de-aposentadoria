# Aula de Planejamento de Aposentadoria (Brasil)

Notebook educacional em PT-BR, cobrindo do basico ao nivel PhD temas de
planejamento de aposentadoria no contexto brasileiro.

## Conteudo
- `demo.ipynb`: aula completa com conceitos, exemplos numericos e graficos.

## Estrutura do notebook
- Camada 1: iniciante (conceitos e contas basicas)
- Camada 2: intermediario (cenarios, sensibilidade e riscos)
- Camada 3: avancado (retiradas, glide path, LDI, sequencia)
- Camada 4: PhD (modelos estocasticos, risco de cauda, fatores, longevidade)

## Requisitos
- Python 3.10+ (ou similar)
- Bibliotecas: `numpy`, `pandas`, `matplotlib`

Instalacao rapida:
```
pip install numpy pandas matplotlib
```

## Como executar
1) Abra `demo.ipynb` no VS Code (extensao Jupyter) ou Jupyter Lab/Notebook.  
2) Rode as celulas na ordem para gerar os graficos.

## Dados e premissas
- O notebook usa snapshots fixos e sinteticos para reproducibilidade.
- Para analises reais, substitua por series oficiais do IBGE/BCB e regras vigentes.

## Aviso
Material estritamente educacional. Nao constitui recomendacao individual.
